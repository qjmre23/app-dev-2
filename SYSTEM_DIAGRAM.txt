                         COMPONENT INTERACTIONS
═══════════════════════════════════════════════════════════════════════════════

MOBILE APP FLOW:
    User Action → Provider → Service → HTTP/WS → Backend → Response
                                                               ↓
                                                    Update UI ← Provider

BACKEND FLOW:
    HTTP Request → Router → Handler → Database → Response
                                         ↓
                              WebSocket Broadcast → All Clients

ARDUINO FLOW:
    RFID Scan → Serial Input → Parse UID → HTTP POST → Backend
                                                ↓
                                          LED Update

DASHBOARD FLOW:
    WebSocket Message → Parse JSON → Update Data → Render UI
